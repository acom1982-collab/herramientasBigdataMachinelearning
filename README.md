# 📊 Proyecto Data Lake: Departamentos de Colombia

Este proyecto implementa un flujo de datos (ETL) utilizando **Docker**, **SQL Server** y **DBeaver**. Se realizó la ingesta de datos brutos, análisis de tipos de datos y carga final en un esquema optimizado.

---

## 🛠️ Tecnologías
* **Motor:** Microsoft SQL Server 2022 (Docker).
* **Herramientas:** DBeaver, Docker Compose.
* **Lenguaje:** T-SQL.

---

## 🚀 Cómo ejecutar
1. Clona el repositorio.
2. En la terminal ejecuta: `docker compose up -d`.
3. Conéctate vía DBeaver:
   - **Puerto:** 1434
   - **User:** sa
   - **Pass:** SuperPWD123!

---

## 📂 Contenido del Proyecto
* `docker-compose.yml`: Configuración del servidor.
* `consultas.sql`: Scripts de análisis y transformación (ETL).
* `README.md`: Documentación.
